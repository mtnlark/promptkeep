## Installation

```bash
# Install using pip
pip install promptkeep

# Or install from source
git clone https://github.com/yourusername/promptkeep.git
cd promptkeep
pip install -e .
```

## Quick Start

1. Initialize your prompt vault:
```bash
promptkeep init ~/PromptVault
```

2. Start adding and managing your prompts!
